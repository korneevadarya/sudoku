# 🧩 KivySudoku

Простая и стильная игра в Судоку, написанная на **Python** с использованием графического фреймворка **Kivy**. Подходит для тренировки логики, изучения принципов работы с Kivy или просто для приятного времяпрепровождения.

## ✨ Возможности
- 🎚️ **Три уровня сложности**: Easy, Medium, Hard
- ⏱️ **Встроенный таймер**: отслеживание времени прохождения
- 🖱️ **Удобный ввод**: поддержка клавиатуры и мыши
- 💡 **Подсветка чисел**: автоматическое выделение ячеек с совпадающими цифрами
- 🏆 **Экран победы**: поздравление и возможность начать новую игру

## 🛠️ Требования
- Python `3.8+`
- Kivy `>= 2.1.0`
- Файл шрифта `label_font.ttf` (должен лежать в корневой папке проекта)
